
# **WSL（Ubuntu）の基本操作まとめ（実務向け）**

## 1. **WSL を起動する**

Windows の PowerShell / コマンドプロンプトで：

コード

```
wsl
```

特定のディストリビューションを起動：

コード

```
wsl -d Ubuntu
```

## 2. **WSL の状態確認**

### インストール状況

コード

```
wsl --status
```

### ディストリビューション一覧

コード

```
wsl --list --verbose
```

## 3. **WSL の終了（安全に停止）**

### 全ての WSL を停止

コード

```
wsl --shutdown
```

### Ubuntu だけ終了（ターミナルを閉じるだけでもOK）

コード

```
exit
```

# Ubuntu につなぐ正しいコマンド

あなたの環境では、ディストリビューション名は **Ubuntu** なので、こうする：

コード

```
wsl -d Ubuntu
```

これで Ubuntu に入れる。

# 🧭 もっと簡単な方法（おすすめ）

PowerShell でただ：

コード

```
wsl
```

と打つだけでも、**既定のディストリビューション（あなたの場合 Ubuntu）に自動で接続**される。

あなたの `wsl --list --verbose` の出力：

コード

```
* Ubuntu                 Running         2
```

→ 先頭に `*` が付いているので **Ubuntu がデフォルト**。

だから：

コード

```
wsl
```
